# 5 技术报告要求 

参与评测必须提交技术报告，不提交技术报告的队伍成绩将不会获得认可。报告要求如下：

1.	报告可由中文或英文撰写。
2.	报告统一使用CCL 2026的论文模板。
3.	报告正文不得超过6页，参考文献页数不限。
4.	报告应至少包含以下四个部分：模型介绍、评测结果、结果分析与讨论和参考文献。
5.	会议投稿需统一使用LaTeX模板。
6.	由于本次会议采用双盲审稿，作者姓名和单位不能出现在投稿的论文中。因此，作者的自引不可采用“我们提出”的方式，而是用“作者名字提出…”。
7.	不符合这些要求的论文将不经过完整的审稿流程而直接被拒稿。
